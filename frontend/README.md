# LifeRemind Frontend

React 18 + Tailwind CSS single-page application.

## Pages
- **Today** — Dashboard with voice input, today's reminders, and habits
- **Calendar** — Month view with reminder indicators
- **Habits** — Habit list with streaks and daily progress
- **Reminders** — All reminders with search and status filters
- **Archive** — Completed reminders history

## Scripts
```bash
npm start    # Dev server on http://localhost:3000
npm run build  # Production build
```

## Environment Variables
```
REACT_APP_API_URL=http://localhost:5000/api
```
